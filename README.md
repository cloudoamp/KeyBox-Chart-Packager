# KeyBox-Chart-Packager
さしみ。さんの制作している音ゲー「KeyBox」の創作譜面作成補助アプリ<br>
<br>
KeyBoxの創作譜面をみんな作ってほしいから補助アプリ作っちゃいました<br>
みんなで作ろう！！！<br>
そしてみんなで共有しよう！！<br>

# 使い方

**基本情報**<br>
曲名:曲名を入れる<br>
作曲者名:作曲者名(feat.なども)<br>
meta ID:よくわからないなら0<br>
BPM:その曲の初めのBPM　MIDIを読み込んだなら自動で読み込まれる<br>
Offset:オフセット<br>
Volume:音量　1より大きい場合は少しずつ大きくなる<br>
DemoStart:選曲画面で流れるデモ視聴の初め(秒)<br>
DemoStart:選曲画面で流れるデモ視聴の終わり(秒)<br>

**元ファイル選択**
MIDIファイル:任意ではありますが、MIDIを読み込むことで、MIDIのなるタイミングにノーツを自動で打ち込んでくれます<br>
音声ファイル:楽曲の音声ファイルを選択します(現時点ではwavのみ)<br>
背景画像:楽曲のジャケットを選択します(現時点ではpng)<br>
<br>

**難易度設定**
譜面製作者名を共通化する:譜面製作者名をすべての難易度で共通化します<br>
BoxPower:KeyBoxの難易度設定<br>
<br>

**出力設定**
KeyBoxのSettings→Miscのデータ保存場所に書いてある場所です。<br>
KeyBox_Dataっていうフォルダだと思います<br>
